# GRYD — Règlement Saison 0

## Objectif

Définir les règles officielles de la première saison publique de GRYD.

La Saison 0 sert à valider :
- la boucle course → capture ;
- l’attachement au territoire ;
- la rétention ;
- l’usage des crews ;
- l’anti-triche ;
- la monétisation non pay-to-win.

Nom conseillé :

```txt
Saison 0 — Fondateurs
```

---

# 1. Durée

Durée recommandée :

```txt
8 semaines
```

Phases :
1. Pré-saison : 7 jours
2. Saison active : 8 semaines
3. Gel des scores : 24 h
4. Résultats : J+1
5. Intersaison : 7 jours
6. Reset carte : au lancement Saison 1

---

# 2. Ce qui est reset

À la fin de la saison :
- propriété des hexes ;
- locks ;
- boucliers ;
- classement saison ;
- score territoire saison ;
- zones contestées ;
- offensives actives.

---

# 3. Ce qui reste

Le joueur conserve :
- compte ;
- pseudo ;
- niveau permanent ;
- badges ;
- titres ;
- skins ;
- médailles ;
- objets collection ;
- posters ;
- historique ;
- records ;
- réputation ;
- statistiques performance ;
- crew, sauf changement volontaire.

---

# 4. France entière ouverte

La Saison 0 couvre toute la France.

Règle :

```txt
Chaque course valide en France peut capturer des hexes.
```

Mais les zones ont des statuts différents :
- active ;
- émergente ;
- pionnière ;
- sauvage.

Les fonctionnalités de guerre avancées sont activées selon la densité locale.

---

# 5. Conditions de validité d’une course

Une course doit respecter :
- distance minimale ;
- durée minimale ;
- allure compatible ;
- GPS suffisamment fiable ;
- cohérence mouvement / pas / vitesse ;
- source acceptable ;
- absence de signaux forts de véhicule.

Statuts possibles :
- valid ;
- partial ;
- flagged ;
- rejected.

Seules les parties validées peuvent capturer.

---

# 6. Capture

Règle :

```txt
Une course capture uniquement les hexes réellement traversés.
```

Pas de capture automatique d’intérieur de boucle.

Types :
- hex neutre ;
- hex volé ;
- hex défendu ;
- hex pionnier.

---

# 7. Points

Barème Saison 0 recommandé :

```txt
+10 pts par hex neutre
+15 pts par hex volé
+3 pts par hex défendu, max 1 fois / 24 h / hex
+5 pts par hex pionnier en zone dense
+8 pts par hex pionnier en zone émergente
+10 pts par hex pionnier en zone sauvage
```

Bonus performance :

```txt
plafonné à +15 %
```

---

# 8. Locks

Chaque hex capturé est verrouillé.

Règle recommandée :

```txt
lock 24 h
```

Pendant le lock :
- l’hex ne peut pas être volé ;
- le propriétaire peut le défendre ;
- le lock est visible sur la carte ;
- le lock ne peut pas être acheté ou supprimé.

---

# 9. Decay

Un hex non repassé depuis une certaine durée redevient neutre.

Durée recommandée :

```txt
21 jours
```

Notification :
- J-3 ;
- J-1 si zone importante ;
- digest si plusieurs zones menacées.

---

# 10. Boucliers

Bouclier de quartier :
- protège un cluster ;
- durée 48 h ;
- visible par tous ;
- non cumulable à l’infini ;
- maximum 2 actifs par joueur/semaine ;
- taille max recommandée : 300 hexes.

Interdit :
- bouclier illimité ;
- bouclier national ;
- bouclier invisible ;
- suppression payante du lock adverse.

---

# 11. Crews

Taille Saison 0 :
- minimum 2 membres ;
- maximum 10 membres.

Règles :
- changement de crew limité ;
- cooldown recommandé : 7 jours ;
- pas de changement de crew pendant les dernières 72 h de saison, ou contribution verrouillée à l’ancien crew ;
- un seul crew actif par joueur.

---

# 12. Classements

Classements recommandés :
- joueur local ;
- crew local ;
- national équilibré ;
- pionnier ;
- performance crew ;
- secteur / quartier ;
- avant-postes.

Le classement national ne doit pas être uniquement brut.  
Il doit équilibrer territoire, exploration, régularité et contribution.

---

# 13. Égalités

En cas d’égalité :
1. nombre de courses valides ;
2. nombre de jours actifs ;
3. hexes défendus ;
4. participation crew ;
5. ancienneté de la première capture ;
6. si toujours égalité : égalité assumée.

---

# 14. Anti-triche

Si course suspecte :
- stats privées possibles ;
- claims gelés ;
- pas de classement ;
- revue admin.

Sanctions :
1. segments exclus ;
2. avertissement doux ;
3. claims gelés ;
4. restriction temporaire ;
5. wipe saison ;
6. ban capture.

Pas de dotations réelles en Saison 0.

---

# 15. Récompenses Saison 0

Récompenses possibles :
- badge Fondateur ;
- skin Fondateur ;
- poster de saison ;
- médaille classement ;
- titre local ;
- titre pionnier ;
- titre crew ;
- bannière profil ;
- carte partage exclusive.

Pas de cash prize.  
Pas de loterie.  
Pas de récompense physique forte avant anti-triche V1.

---

# 16. Règles de modification

Interdit de changer pendant la saison :
- barème de points ;
- durée lock ;
- durée decay ;
- plafond bonus performance ;
- règles de bouclier ;
- critères de classement.

Modifications autorisées :
- correction bug ;
- amélioration anti-triche ;
- clarification UX ;
- patch de sécurité.

Chaque modification doit être publiée en patch notes.

---

# 17. Contestations

Un utilisateur peut demander une revue si :
- sa course n’a pas capturé ;
- ses points semblent incorrects ;
- ses segments ont été exclus ;
- son compte est flaggé.

Limites :
- nombre de revues gratuit limité ;
- les abus de contestation peuvent être bloqués ;
- décision admin finale pour Saison 0.

---

# 18. Règlement public simplifié

Version courte affichable :

```txt
Cours réellement.
Reste sur les voies accessibles au public.
Ne triche pas avec vélo, voiture ou faux GPS.
Ne harcèle personne.
Ne partage pas d’information privée.
La carte reset à chaque saison.
Ton niveau, tes badges et tes récompenses restent.
```
