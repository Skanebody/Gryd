# GRYD — Sponsors, partenariats, marques et revenus B2B

## Objectif

Définir la vision sponsor/partenaires de GRYD après validation produit.

À ne pas surcharger au MVP, mais à anticiper.

GRYD peut devenir un média local + running + jeu.

---

# 1. Principe

Les sponsors ne doivent jamais acheter la carte au détriment des joueurs.

Interdit :
- sponsor qui donne des points ;
- sponsor qui achète un territoire ;
- sponsor qui favorise un crew en classement ;
- sponsor qui force un parcours dangereux.

Autorisé :
- challenges ;
- stations virtuelles ;
- récompenses cosmétiques ;
- dossards ;
- coupons ;
- événements ;
- cartes sponsorisées clairement identifiées.

# 3. Formats sponsorisables

## Challenge sponsorisé

Exemple :
```txt
Challenge On Running — 50 km en 14 jours
```

Récompenses :
- badge ;
- skin ;
- tirage optionnel légalement validé ;
- réduction partenaire.

## Station virtuelle

Point sur carte :
- ravito ;
- checkpoint ;
- boutique partenaire ;
- zone photo.

Ne doit pas donner d’avantage compétitif fort.

## Skin sponsorisé

Exemple :
- skin trace On ;
- badge Nike Run ;
- poster Race Day.

## Dossards partenaires

Après anti-triche solide :
- gagnés par mission ;
- pas loterie mal cadrée ;
- règlement clair.

## Événement crew

Exemple :
```txt
Week-end Crew Battle Paris
```

---

# 4. Sponsoring local

Très intéressant pour zones actives.

Exemples :
- magasin running sponsorise un secteur ;
- café runner devient station ;
- club local organise une offensive officielle ;
- partenaire offre récompense aux crews actifs.

Attention :
- signalétique commerciale claire ;
- pas de pression à entrer dans un lieu ;
- pas de tracking sponsor individuel.

---

# 5. Données sponsor

Ne jamais vendre :
- traces individuelles ;
- données santé ;
- adresse ;
- zones privées ;
- comportement individuel identifiable.

Possible :
- statistiques agrégées anonymisées ;
- nombre de participants ;
- km cumulés ;
- zones actives ;
- impressions sponsor ;
- participations challenge.

Prix indicatif :
- 500-1 500 €

- visibilité carte ;
- post social.

## City Partner

Prix :

Contenu :

- skins ;

## Brand Season

Prix :

- collection skins ;
- challenge national ;
- activation run clubs.

# 7. Timing

## MVP

Pas de sponsor.
Objectif : valider rétention.

## Saison 1

Sponsors locaux légers :
- dossards ;
- magasins running ;
- challenges sans avantage classement.

## Saison 2

Marques running :
- collections ;
- événements ;
- challenges nationaux ;
- stations virtuelles.

---

# 8. Garde-fous

1. Transparence sponsor.
2. Pas de pay-to-win sponsorisé.
3. Pas de données individuelles.
4. Pas d’incitation dangereuse.
5. Règlement clair.
6. Dotations uniquement après anti-triche.
7. Validation juridique.

---

GRYD transforme chaque run en conquête territoriale.
