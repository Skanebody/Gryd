# PROMPT — GRYD — ÉCONOMIE IN-APP, MONÉTISATION ET BOUCLES PAYANTES INSPIRÉES SUPERCELL

## Contexte produit

Tu es un **Game Economy Designer senior + Product Monetization Lead + spécialiste free-to-play mobile**.

Je construis **GRYD**, une app mobile de conquête de territoire par la course à pied.

Concept :
- L’utilisateur court dans sa ville.
- Sa trace GPS capture des hexagones sur une carte réelle.
- Les joueurs peuvent rejoindre des **crews**.
- Les crews cumulent leur territoire.
- Les autres crews peuvent reprendre les zones.
- Le jeu fonctionne par saisons.
- Les utilisateurs gagnent XP, Foulées, badges, skins, titres et récompenses.
- Le jeu doit monétiser dès le lancement sans devenir pay-to-win.

Nom public : **GRYD**  
Baseline : **Cours. Capture. Défends.**  
Positionnement : **Le jeu de conquête de territoire pour run clubs.**

## Références à utiliser

Inspire-toi fortement de :
- **Clash of Clans** ;
- **Brawl Stars** ;
- **Clash Royale** ;
- **Hay Day** ;
- la philosophie Supercell.

Mais ne copie pas mécaniquement.

Ce qu’il faut reprendre de Supercell :
- progression permanente ;
- pass saisonnier ;
- monnaie premium ;
- packs contextuels ;
- offres limitées ;
- valeur perçue ;
- communauté/clan/crew ;
- attachement aux ressources ;
- accélération sans destruction de l’équité ;
- rendez-vous quotidiens ;
- évènements saisonniers ;
- statut social ;
- rareté cosmétique.

Ce qu’il ne faut pas faire :
- vendre la victoire ;
- vendre directement le territoire ;
- vendre des points de classement ;
- vendre des kilomètres ;
- créer une expérience casino ;
- frustrer artificiellement un utilisateur qui fait du sport.

## Règle absolue

GRYD ne doit jamais devenir pay-to-win.

L’argent peut acheter :
- confort ;
- information ;
- protection temporaire et plafonnée ;
- statut ;
- cosmétique ;
- progression hors classement ;
- accélération de craft cosmétique ;
- accès premium ;
- personnalisation crew ;
- archives/posters ;
- templates de partage.

L’argent ne peut jamais acheter :
- hexes ;
- kilomètres ;
- points de classement ;
- vols automatiques ;
- suppression du lock ;
- boucliers illimités ;
- invisibilité ;
- domination directe ;
- victoire de saison.

# Économie de base

Propose une économie avec au minimum :

## 1. Points de saison

Utilité :
- classement saison ;
- progression compétitive ;
- reset saisonnier.

Règle :
- non achetables ;
- gagnés uniquement en courant.

## 2. XP joueur

Utilité :
- niveau permanent ;
- titres ;
- statut ;
- déblocage progressif.

Règle :
- peut être boosté par des boosts XP, mais ne doit pas influencer le classement territoire.

## 3. Foulées

Monnaie douce.

Utilité :
- skins earnables ;
- fragments ;
- petits cosmétiques ;
- renommage basique ;
- crafting non premium.

Règle :
- gagnées en courant ;
- bonus Club possible ;
- pas d’achat direct obligatoire.

## 4. Éclats

Monnaie premium.

Utilité :
- skins premium ;
- artefacts ;
- packs ;
- templates ;
- emblèmes ;
- effets visuels ;
- accélération de progression cosmétique.

Règle :
- achetés via in-app purchase ;
- ne doivent pas expirer ;
- ne doivent pas acheter de domination directe.

## 5. Fragments

Ressource de collection/craft.

Utilité :
- débloquer skins ;
- améliorer artefacts ;
- assembler objets ;
- donner une raison de revenir.

Règle :
- principalement gagnés par les courses, coffres gratuits, objectifs crew et pass.
