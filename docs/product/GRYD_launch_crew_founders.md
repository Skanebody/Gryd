# GRYD — Lancement Crews Fondateurs, acquisition et go-to-market

## Objectif

Définir le plan de lancement de GRYD.

GRYD ne peut pas être lancé comme une app vide.  
Il faut créer la densité sociale avant l’ouverture.

Objectif :

```txt
30 à 50 crews fondateurs avant Saison 0.
```

---

# 1. Positionnement lancement

Nom :

```txt
Saison 0 — Fondateurs
```

Message :

```txt
La France ouvre.
Les premiers crews écrivent la carte.
```

# 3. Offre Crew Fondateur

Pack non pay-to-win :
- badge Crew Fondateur ;
- accès TestFlight ;
- 10 invitations ;
- skin crew ;
- emblème ;
- poster saison ;
- page crew publique ;
- titre local si top classement.

Prix possible :
- gratuit pour premiers crews stratégiques ;
- 19,99 € pour création premium ;
- ou Founder Pack individuel à 4,99/9,99 €.

---

# 4. Funnel

## Étape 1 — Waitlist

Landing page :
- ville ;
- code postal ;
- création crew ;
- invitation membres.

## Étape 2 — Qualification

Questions :
- ville ;
- nombre de coureurs ;
- fréquence ;
- run club existant ;
- Instagram / Strava.

## Étape 3 — Onboarding crew

Créer :
- nom crew ;
- emblème ;
- zone principale ;
- leader ;
- lien invitation.

## Étape 4 — TestFlight

Donner accès :
- leader d’abord ;
- puis membres ;
- règles bêta.

## Étape 5 — Saison 0

Lancer avec rivalités déjà créées.

# 5. Script DM Instagram

```txt
Salut [Prénom], je lance GRYD : un jeu de conquête de territoire par la course.
Ton crew court, capture des zones sur la carte, défend ses quartiers et monte au classement.

On ouvre la Saison Fondateurs avec quelques crews seulement.
Je pense que votre groupe peut faire partie des premiers.

Tu veux que je t’envoie l’accès crew ?
```
