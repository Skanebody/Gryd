# Fontes licenciées (AMENDEMENT-03 — typo Outcrowd)

La typo de référence est **ITC Avant Garde Gothic Std** (celle d'outcrowd.io : Md pour titres/UI, Bk pour le texte).
C'est une police commerciale (Monotype ; dispo via Adobe Fonts ou MyFonts) — **ne jamais commiter de fichier non licencié ici**.

Une fois la licence acquise, déposer :
- `ITCAvantGardeStd-Bk.woff2` (400)
- `ITCAvantGardeStd-Md.woff2` (500)
- `ITCAvantGardeStd-Demi.woff2` (700)

puis activer le bloc `localFont` documenté dans `app/layout.tsx`.
En attendant, le site sert **Poppins** (Google), l'alternative libre la plus proche, + **Lora** (exacte, libre).
